\section{Absorption Law}
Absorptive Law â€“ This law allows for the reduction of a difficult statement to a simpler one by absorbing phrases that are similar in structure. 
A + (A.B) = (A.1) + (A.B) = A(1 + B) = A (OR Absorption Law)
A(A + B) = (A + 0)
A(A + B) = A + (0.B) = A (AND Absorption Law)
A(A + B) =A
